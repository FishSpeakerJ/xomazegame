# Main Features #
  * Maze
  * Collaborative
  * Fog of War
  * Head Collection
  * Mutual Start & Exit
  * Color Paths
  * Key to signal other players
    * dance with no head
    * dance AND sound with head
    * drop path when signaling next to other player heads


# Game Sequence #
  * See Maze
  * Fog of War rolls in
  * Heads pop off into the maze
  * START!


# Other Special Features NOT in Game #
  * Arcs
  * Arks
  * Rotation
